# tutedude-Assignment-5

## Task 1: Student Marks Dictionary
- Creates a dictionary of student names and their marks.
- Asks the user to enter a student's name.
- Displays the student's marks if found.
- Shows "Student not found." if the name does not exist.
## Task 2: List Slicing
- Creates a list of numbers from 1 to 10.
- Extracts the first five elements.
- Reverses the extracted elements.
- Prints the original list, extracted list, and reversed list.
